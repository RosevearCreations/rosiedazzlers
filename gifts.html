Gift Certificates

Buy a gift certificate for someone else (or yourself). Choose either a Service Gift (exact package + vehicle size) or a Dollar Gift (amount deducted from any booking).

Valid 12 months from purchase No refunds Service Gifts must match size Dollar Gifts apply to any booking
Important: Gift Certificates are a different system than booking deposits. You purchase here (cart → Stripe Checkout). You redeem later during booking using the Gift Code (RD-XXXXXX...).

How it works

  1. Buy a gift certificate here (Stripe Checkout).
  2. Receive a Gift Code (RD-XXXXXX...).
  3. Redeem the code during booking to apply the service or dollar value.

Rules

  • Valid for 1 year (12 months) from purchase date.
  • No refunds on gift certificate purchases.
  • Service Gift vouchers must match the correct package + vehicle size.
  • Dollar Gift vouchers deduct from the booking total.
If email delivery is not configured yet, keep your Stripe receipt email. If you need help locating a Gift Code, contact us with the purchaser email and date.
