Gift Certificates

Buy a gift certificate for someone else (or yourself). Choose either a Service Gift (exact package + vehicle size) or a Dollar Gift (amount deducted from any booking).

Service Gifts = exact package + size Dollar Gifts = apply amount at booking Valid 12 months from purchase No refunds
Important: Gift Certificates are purchased here using Stripe Checkout. They are redeemed later during booking using the Gift Code (RD-XXXXXX...).

How Gift Certificates work

  1. Buy a gift certificate here (add to cart → Stripe Checkout).
  2. Receive a Gift Code (RD-XXXXXX...).
  3. Redeem the code during booking to apply the service or dollar amount.

Rules

  • Valid for 1 year (12 months) from purchase date.
  • No refunds on gift certificate purchases.
  • Service Gifts must match the package + vehicle size at booking.
  • Dollar Gifts apply as a deduction during booking.
